Projet k-plus proches voisins

Concevoir un classifieur automatique basé sur l'algorithme des k-plus proches voisins. Ce modèle permettra de prédire une caractéristique d'un individu (ou objet) en se basant sur les attributs descriptifs de cet individu, à l'aide d'un jeu d'entraînement. Peut être utilisé dans le but de recommander des produits ou services, segmenter les clients en clusters pour des campagnes marketing adaptées ...

  • ⚙️ Implémentation d'un classifieur automatique en Python
  • 📚 Utilisation de jeux d'entraînement et de test pour la prédiction
  • 🔧 Développement d'un algorithme de classification supervisée
